## Optimization Guidelines:
- Format: WebP
- Quality: 80-85%
- Dimensions: 400x600px (2:3 aspect ratio)
- File size: <50KB each

## Conversion Command:
```bash
# Convert existing images to WebP
cwebp -q 85 input.jpg -o output.webp
```
